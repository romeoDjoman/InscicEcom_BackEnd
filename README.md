# Projet InscicEcom

## Vue d'ensemble

Ce projet est construit avec ****Angular*** pour le frontend et ***Spring Boot*** pour le backend, utilisant ***MySQL*** comme base de données. L'objectif est de créer une expérience d'achat en ligne fluide avec une interface conviviale et un backend robuste.

## Fonctionnalités

- **Authentification des utilisateurs :** Fonctionnalité sécurisée d'enregistrement et de connexion des utilisateurs.
- **Catalogue de produits :** Parcourez un catalogue diversifié de produits avec des informations détaillées.
- **Panier d'achat :** Ajoutez des produits au panier, passez en revue et procédez au paiement.
- **Gestion des commandes :** Consultez l'historique des commandes et suivez l'état des commandes en cours.
- **Panneau d'administration :** Gérez les produits, les utilisateurs et les commandes via une interface d'administration dédiée.

## Technologies utilisées

- **Frontend :**

  - ***Angular*** : Un puissant et dynamique framework frontend pour la création d'applications web réactives.
  - ***HTML/CSS*** : Technologies web standards pour structurer et styliser l'interface utilisateur.
  - ***TypeScript*** : Le langage de programmation utilisé en conjonction avec Angular pour construire des applications évolutives.

- **Backend :**

  - ***Spring Boot*** : Un framework basé sur Java pour la création d'applications de niveau entreprise avec facilité.
  - ***Spring Security*** : Assure une authentification et une autorisation sécurisées des utilisateurs.
  - ***Spring Data JPA*** : Simplifie les interactions avec la base de données grâce à l'API de persistance Java.

- **Base de données :**

  - ***MySQL*** : Un système de gestion de base de données relationnelle pour stocker les données des produits, des utilisateurs et des commandes.

## Pour commencer

1. **Configurer le Backend :**

   - Configurez les paramètres de votre base de données ***MySQL*** dans le fichier `application.properties`.
   - Exécutez l'application Spring Boot.

2. **Configurer le Frontend :**

   - Accédez au répertoire du frontend.
   - Installez les dépendances : `npm install`
   - Démarrez l'application Angular : `ng serve`

3. **Accéder à l'application :**

   - Ouvrez votre navigateur et allez à `http://localhost:4200` pour accéder à l'application ***InscicEcom***.
